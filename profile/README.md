# druthers.io

Social taste-sharing for the media you love — Movies, TV, Books, and Games.
Track what you've watched, played, and read; share a formatted top-5; find the
overlap with a friend.

Built on Cloud Run · Neon Postgres · Cloudflare, with an OSS security pipeline
across every repo (Gitleaks · Semgrep · Trivy · Dependabot).

*This profile is served from the `ALeonard9/.github` repository.*
